beroepsproducten

Garage Wiets

Bij dit project moesten wij een windows server omgeving maken met daarbij een werkstation en een webserver op een linux distributie. Bij dit project ging het om verschillende rechten en snelkoppelingen te geven aan gebruikers op basis van in welke afdeling ze zaten. Daarnaast moesten de afdelingen gescheiden zijn van elkaar.

TeraDruk

Bij dit project moesten wij samen met een klas genoot twee domain controllers opzetten op 2 locaties en deze redundant met elkaar verbinden via een vpn verbinding. Voor deze vpn tunnel hebben wij PFSense gebruikt. Tussen de 2 PFSense routers hebben wij de vpn tunnel gemaakt met IPSec. Daarnaast moest er ook een webserver komen in de DMZ.
